### 🧪 Currently playing with

#### 🧠 AI & Experiments
- **[openenv-0ad-bridge](https://github.com/0xrushi/openenv-0ad-bridge)**: Interface library connecting RL agents to the *0 A.D.* engine for post-training environments.
- **[Hunyuan2Minecraft](https://github.com/0xrushi/Hunyuan2Minecraft)**: Bridging Hunyuan 3D generation models directly into Minecraft.
- **[HeteroShard](https://github.com/0xrushi/HeteroShard)**: An opensource heterogeneous GPU training framework.
- **[speech-dataset-generator](https://github.com/0xrushi/speech-dataset-generator)**: 🔊 Advanced audio processing toolkit to create labeled datasets, enhance quality, and identify speakers.
- **[dailytalk-contiguous-generator](https://github.com/0xrushi/dailytalk-contiguous-generator)**: Pipeline to generate emotion-tagged, contiguous conversational audio (suitable for fine-tuning Moshi).
- **[TTSizer](https://github.com/0xrushi/TTSizer)**: 🎙️ Automatically transcribe audio/video into high-quality, speaker-specific Text-To-Speech datasets.

#### ⚙️ Utilities & Tools
- **[portable-hnsw](https://github.com/0xrushi/portable-hnsw)**: Serverless vector search—serving HNSW indices directly from CDNs to the browser.
- **[rpi-energy-logger](https://github.com/0xrushi/rpi-energy-logger)**: Lightweight Raspberry Pi telemetry logger (CPU, power, processes) using SQLite WAL-mode.
- **[tradingview-scraper](https://github.com/0xrushi/tradingview-scraper)**: Automated tool for extracting charts and metrics from TradingView.
